# Contributing to RecipeBook

## Recipe Format Requirements

All recipes must include:
- Title (ALL CAPS at top)
- Prep time
- Cook time
- Servings
- Ingredients (each starting with -)
- Instructions (numbered steps)


## Pull Request Process
1. Create a branch named 'recipe/your-recipe-name'
2. Add your recipe file to the recipe/ folder
3. Update README.md to include your recipe
4. Submit Pull Request
